# Cidadania Digital: Prevenção de Golpes 🛡️

**Acesse o jogo online aqui:** https://alemendesva.github.io/CidadadaniaDigital/

**Versão:** 1.0.0

## 🎯 Qual problema estamos tentando resolver?
Muitos idosos e pessoas com pouca vivência tecnológica são alvos diários de golpes digitais, engenharia social e furtos de credenciais. A ausência de letramento digital gera vulnerabilidade financeira e emocional. 

## 👥 Público-alvo e Solução
Desenvolvido com foco no público 60+, este projeto é um sistema educativo (Web GUI) de **Prevenção de Golpes**. Utilizando histórias interativas e um ambiente livre de jargões técnicos complexos, a aplicação simula cenários reais (como senhas anotadas em post-its ou o gatilho da urgência em compras) e testa o conhecimento do usuário por meio de um quiz de fixação de aprendizado.

## ⚙️ Tecnologias Utilizadas
- **Interface e Lógica:** HTML5, CSS3, JavaScript (Vanilla).
- **Testes Automatizados:** Jest.
- **Análise Estática (Linting):** ESLint.
- **CI / Pipeline:** GitHub Actions.
- **Gerenciador de Dependências:** NPM (`package.json`).

## 🚀 Como instalar e executar

### 1. Executando a Aplicação (Interface)
Basta clonar o repositório e abrir o arquivo `index.html` em qualquer navegador moderno. Nenhuma instalação de servidor é necessária para o uso principal.

### 2. Executando as ferramentas de Qualidade (Lint e Testes)
Para rodar os testes e o linting localmente, você precisará do [Node.js](https://nodejs.org/) instalado.
